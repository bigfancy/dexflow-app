The next generation decentralized exchange platform with advanced features and seamless trading experience.

## Features

- DFT Token: Native platform token
- DFNFT: NFT creation and management
- DEX: Fast and efficient token swaps
- Liquidity Pools: Provide liquidity and earn rewards
- NFT Auctions: Both English and Dutch auction mechanisms
- On-chain Advertising: Innovative advertising platform

## Quick Start

1. Install dependencies:
```bash
yarn install
```

2. Set up environment:
```bash
cp .env.example .env
```

3. Start local network:
```bash
yarn chain
```

4. Deploy contracts:
```bash
yarn deploy
```

5. Start NextJS app:
```bash
yarn start
```

6. Run click settlement cron:
```bash
yarn cron
```

## Technical Stack

- Frontend: Next.js, TypeScript, TailwindCSS
- Smart Contracts: Solidity
- Testing: Hardhat
- Database: Prisma
- Deployment: Vercel

## Development

The project follows a monorepo structure:
- `/packages/hardhat`: Smart contracts and deployment scripts
- `/packages/nextjs`: Frontend application
- `/packages/common`: Shared utilities and types
