# Setup
1.Install dependencies if it was skipped in CLI:
```
yarn install
```

2. Set up environment variables:
```
cp .env.example .env
```

3. Run a local network
```
yarn chain
```

4. Deploy contracts
```
yarn deploy
```

5. Run NextJS app
  ```
  yarn start
  ```

6. Run the cron job to settle clicks:
  ```
  yarn cron
  ```
